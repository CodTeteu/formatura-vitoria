# Convite Template

Template reutilizavel de convites digitais para formatura e casamento, com RSVP, painel admin, backend Vercel Functions e Supabase por convite.

Consulte `docs/SETUP.md`, `docs/NOVO-CONVITE.md`, `docs/PROMPT-VIBECODING.md` e `docs/QA-CHECKLIST.md` antes de criar um novo convite.
